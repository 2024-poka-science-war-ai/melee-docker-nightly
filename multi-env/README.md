# super-smash-bros-melee-reinforcement-learning


## Installation

```bash
pip3 intall -e libmelee
pip3 intall -e melee-env
```

## Running the Example Agent

```bash
python3 melee-env/agents_example.py --iso ./ssbm.iso
```
